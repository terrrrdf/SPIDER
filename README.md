# SPIDER: Soft-constraint Physics-Informed Diffusion model for Embankment Reconstruction

This repository accompanies the manuscript:
“Soft-constraint physics-informed diffusion models for pore water pressure field reconstruction in embankment dams using sparse sensors.”

## Release plan
The source code and the dataset used in this study will be made publicly available upon publication of the paper.

## Contents
This repository will include:
- Training and inference code for SPIDER
- Configuration files and scripts to reproduce the main experiments
- Documentation for running the model and using the dataset
